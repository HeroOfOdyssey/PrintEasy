# Client Moved

The ESP32 Arduino reference client now lives in:

```text
../clients/esp32-arduino/
```

New client targets are organized under `../clients/`:

* `esp32-arduino/` - ESP32 Wi-Fi to Bluetooth Classic SPP printer bridge.
* `linux-python/` - Raspberry Pi and Linux bridge for serial, raw USB, and RFCOMM Bluetooth devices.
* `esp8266-arduino/` - ESP8266 Wi-Fi to UART/SoftwareSerial printer bridge starter.
* `pico-w/` - Pico W / Pico 2 W Wi-Fi to UART printer bridge starter.
* `protocol/` - Shared MQTT payload contract for all clients.
