# Legacy Client Path

Use the multi-client tree for device firmware and bridge daemons:

```text
../clients/
```

Available targets:

* `esp32-arduino/` - ESP32 Wi-Fi to Bluetooth Classic SPP printer bridge.
* `linux-python/` - Raspberry Pi and Linux bridge for serial, raw USB, and RFCOMM Bluetooth devices.
* `esp8266-arduino/` - ESP8266 Wi-Fi to UART/SoftwareSerial printer bridge.
* `pico-w/` - Pico W / Pico 2 W Wi-Fi to UART printer bridge guide.
* `protocol/` - Shared MQTT payload contract for all clients.
